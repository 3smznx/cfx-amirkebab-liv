# cfx-amirkebab-liv

## 🍢 Amir Kebab Food Truck Livery

A high-quality custom **Amir Kebab** livery for the **Gabz Food Truck (`gbboxboyft`)** in FiveM.

This resource features a unique and detailed food truck design with custom branding, menu graphics, and a clean halal-inspired theme, making it perfect for roleplay servers.

## ✅ Compatibility

- ✔ Compatible with **Gabz Food Truck (`gbboxboyft`)**
- ✔ Optimized for FiveM
- ✔ Ready to use
- ✔ High-quality textures

## 📦 Installation

1. Download this resource.
2. Make sure you have the **Gabz Food Truck (`gbboxboyft`)** installed.
3. Place this resource inside your `resources` folder.
4. Add the following line to your `server.cfg`:

```cfg
ensure cfx-amirkebab-liv
```
